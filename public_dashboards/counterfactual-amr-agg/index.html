Counterfactual Equations

反事实推演方程

这一部分把“模型怎么写成方程”补完整。这里不是重新建一个普通 OLS，而是沿用已筛选固定效应模型的估计系数与固定效应项，再把指定气候变量替换为基准期值，得到反事实预测。

1. 固定效应主方程

y_it = Σ_k (β_k · X_kit^(z)) + FE_i + FE_t + ε_it

其中，y_it 是 AMR_AGG_z；X_kit^(z) 是进入当前模型的标准化解释变量；FE_i 表示省份固定效应，FE_t 表示年份固定效应。对于仅年份固定效应模型，仅保留 FE_t；对于双向固定效应模型，同时保留二者。本页当前输出的基准期是 2014。

2. 反事实替换方程

ŷ_it^cf(s) = Σ_(k∉S_s) (β̂_k · X_kit,actual^(z)) + Σ_(k∈S_s) (β̂_k · X_ki,base^(z)) + FÊ_i + FÊ_t

S_s 是情景 s 下被恢复到基准期的变量集合，例如“仅 R1xday 恢复基准”或“所有气候变量恢复基准”。X_ki,base 取该省在基准期内的均值，其他变量保持实际观测值不变。

3. 结果差值与相对变化

Δ_it^(s) = ŷ_it^actual - ŷ_it^cf(s) RelChange_it^(s) = 100 × Δ_it^(s) / |ŷ_it^cf(s)|

当 Δ_it^(s) > 0 时，表示相对于“恢复基准”的世界，实际气候轨迹对应更高的预测 AMR。网页中的全国年度平均、分省长期平均和最新年份结果，都是在这个差值基础上继续汇总得到的。

4. 汇总指标怎么来

全国年度平均: Δ̄_t^(s) = (1 / N_t) · Σ_i Δ_it^(s) 分省长期平均: Δ̄_i^(s) = (1 / T_i) · Σ_t Δ_it^(s)

所以这页上你看到的时间序列图、分省地图和模型比较图，实质上都是对 Δ_it^(s) 在不同维度上的再汇总。

Interpretation

整体结果分析

这里把“哪些结论较稳、哪些对模型设定敏感、R1xday 和抗菌药物使用强度在主模型里处于什么位置”直接写成可用于结果段落的解释。

1. 主模型为何能承接正文

方案A_平衡主线组 在 Year FE 下同时保留了 R1xday 和 抗菌药物使用强度 的正向信号，最适合承接“气候相关暴露在既有抗菌药物使用背景下提高 AMR 风险”的主叙事。

2. 哪些结果较稳

“所有气候变量恢复基准”情景在 4 个入选模型中全部为正差值，范围为 +0.020 到 +0.109。总体气候负担方向较稳。

3. 哪些结果更敏感

“仅 R1xday 恢复基准”在 3 个 Year FE 模型中都为正，但双向 FE 的差值仅为 -0.000；“仅温度变量恢复基准”在主模型中接近 0，但在另外两个 Year FE 稳健性模型中明显为正。

4. 双向 FE 应如何解释

双向 FE 口径下，主模型同变量集的 R1xday 系数已收缩为 -0.003，说明它更适合作为保守下界，而不是正文中的主量化入口。

5. 分省异质性如何写

在主模型主情景下，长期平均正差值较高的省份主要包括 上海, 天津, 北京, 江苏, 辽宁；负差值较高的省份主要包括 海南, 重庆, 福建, 湖南, 贵州。到 2023 年，正差值较高的省份主要转向 北京, 上海, 河北, 河南, 吉林，负差值较高的省份则集中在 海南, 安徽, 湖南, 贵州, 云南。这说明全国平均结果可以支持总体 burden 的存在，但空间分布具有明显异质性。
