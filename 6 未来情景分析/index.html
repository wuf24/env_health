6 未来情景分析

把未来情景结果写成一页能读清的研究报告。

这版不再把页面做成“先切选项再自己找结论”的仪表盘，而是按真正的阅读顺序重排： 先讲最关键的判断，再把两种 baseline 的全国结果摆在一起，然后往下看地区、再看省级，最后才回到稳健性和方法来源。 页面里所有已产出的主要图件都直接展开，不需要另外打开文件。

先看什么

如果你只想抓结论，先看“Key Judgements”和“全国结果”。如果你在写结果部分，再继续往下看地区和省级空间格局。

当前边界

当前 SSP 情景真正进入未来路径的协变量只有 R1xday；因此两种 baseline 的情景增量几乎相同，主要差异来自 baseline 水平本身。

怎么理解稳健性

主模型、低 VIF、systematic、strict FE 不是四个装饰性标签，而是四套不同历史系数。因为未来只让 R1xday 变动，所以 R1xday 系数大小直接决定了情景敏感度。

Key Judgements

这页最重要的判断，不需要先读表就能抓到。

这四条是当前这版结果里最值得先说清的事实。它们决定了后面怎么解释全国曲线、怎么解释空间差异，也决定了应该把哪一版 baseline 放主分析、哪一版放扩展分析。

1. baseline drives level
2.211

X-driven 比 Lancet ETS 在 2050 高出 2.211 个点，而两者在 2024 只差 0.128。这说明两版最大的差别不是情景增量，而是 baseline 怎么延伸到 2050。

2. scenario spread is modest
0.129

主模型下，SSP5-8.5 与 SSP1-1.9 的 national delta spread 只有 0.129。这意味着当前由 R1xday 驱动的 SSP 分歧存在，但没有大到压过 baseline 趋势。

3. spatial pattern is split
华南

SSP5-8.5 下，地区层面最高是 华南 +1.61，最低是 华中 -0.63。全国平均掩盖了明显的区域分化。

4. province heterogeneity is large
海南

SSP5-8.5 下省级抬升最高是 浙江，最低是 河南；而 SSP5-8.5 与 SSP1-1.9 的最大 gap 出现在 海南。

Baseline Logic

先把两种 baseline 分清，再去看结果，结论才不会串。

两种 baseline 共享同一套历史 FE 系数、同一套未来 R1xday 情景和同一套全国平均口径。它们唯一真正不同的地方，是“未来 baseline 到底由什么生成”。

Lancet ETS

先让 AMR 自身沿 ETS 延续

这版最贴近 Lancet 2023 的表述：baseline scenario continued at current rates, as estimated by ETS models。它更强调结果变量自身历史惯性。

Y_it = α_i + λ_t + Σ β_k Z_itk + ε_it Y^base_it = ETS(Y_i, historical series) Δ^scenario_it = Σ β_k × (Z^scenario_itk - Z^base_itk) Y^scenario_it = Y^base_it + Δ^scenario_it
  • 2024 到 2050 的 national baseline 从 30.79 下降到 27.21。
  • 适合作为主分析，因为它最容易和参考文献对齐。
  • 缺点是 AMR 历史惯性更强，未来曲线更容易继续往历史方向延长。
X-driven

先用协变量路径重建 baseline

这版更接近“由未来协变量路径驱动未来结果”的解释框架。它不是完整的 Nature Medicine 时空贝叶斯模型，但逻辑上更靠近 X-driven baseline。

Y_it = α_i + λ_t + Σ β_k Z_itk + ε_it Y^base_it = α_i* + λ_t* + Σ β_k Z^base_itk Y^scenario_it = α_i* + λ_t* + Σ β_k Z^scenario_itk If only R1xday varies: Y^scenario_it = Y^base_it + β_R × (R1xday^scenario_it - R1xday^base_it)
  • 2024 到 2050 的 national baseline 从 30.91 下降到 29.42，下降幅度小于 Lancet ETS。
  • 更适合回答“如果未来气候路径真的分化，AMR 会被推开多少”这类机制问题。
  • 当前和 Lancet ETS 的情景增量一致，说明两版差异主要来自 baseline 水平，而不是 scenario adjustment。

Spatial Results

真正决定“全国平均长什么样”的，是空间分异，而不是一条全国线本身。

七大区和省级结果都显示出非常明显的空间异质性。当前因为两种 baseline 的 scenario delta 相同，所以空间 pattern 基本一致，差异仍主要体现在 baseline level。

地区层面的统一结论

  • 在主模型下，SSP1-1.9 时 7 大区中有 4 个正值、3 个负值；到 SSP5-8.5 变成 5 个正值、2 个负值。
  • 华南始终是最高增量区，到 SSP5-8.5 为 +1.61；华中始终最低，到 SSP5-8.5 为 -0.63。
  • 这意味着全国均值上看似温和的变化，实际上是由一部分地区显著抬升、另一部分地区继续走低共同拼出来的。

省级层面的统一结论

  • 在 SSP5-8.5 下，31 省里有 20 个省为正、11 个省为负，省际方向并不一致。
  • 抬升最高的是 浙江 +2.28，下降最低的是 河南 -2.49。
  • 双情景 gap 最大的省份是 海南，说明它对未来气候路径分化最敏感。
Robustness

稳健性不是附录装饰，它直接告诉你情景敏感度到底是谁在驱动。

稳健性结果里，主模型和低 VIF 模型几乎重合；systematic 模型更敏感，strict FE 几乎被压平。这和 R1xday 系数完全一致：主模型 0.869、低 VIF 0.873、systematic 1.064，而 strict FE 只有 0.042。

Appendix

方法边界、运行命令和关键文件入口。

当前结果的解释边界

  • 当前 SSP 情景真正接入未来路径的协变量只有 R1xday，因此这版更像是在回答“未来极端降水路径会把 AMR 推开多少”。

  • 如果后续把这些外部输入补齐，这个页面的框架仍然适用，只需要把 narrative 里的“仅 R1xday 变化”改掉。
