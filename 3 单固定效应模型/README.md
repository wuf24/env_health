# 3 单固定效应模型

这个目录负责把综合指标后面的 `13` 个 AMR 指标拆开，分别运行固定效应模型，用来回答：

- 综合指标 `AMR_AGG_z` 的信号是否能在单菌种层面被普遍复制？
- 哪些菌种对气候、污染、卫生或发展代理更敏感？
- 主结果是否主要由少数菌种驱动？

它的定位更接近“异质性核对”和“结果拆解”，而不是当前论文主结果的第一入口。

## 这个目录在整条链里的角色

- `2 固定效应模型/` 给出综合指标 `AMR_AGG_z` 的主筛选结论；
- `3 单固定效应模型/` 检查这种综合信号在单菌种上有没有普遍复制；
- 因此它更像对主结论的“拆解验证层”。

如果你现在在写正文主结果，优先看 `2 固定效应模型/`；  
如果你要解释异质性、补充材料或回应“是不是某几个菌种在驱动综合结论”，再回到这里。

## 当前目录结构

- `1_VIF.ipynb`
  - 检查解释变量之间的多重共线性。
- `2_single_species.ipynb`
  - 单菌种固定效应主流程，对 `13` 个 AMR 指标逐个建模。
- `3_selected_X.ipynb`
  - 在单菌种分析中只保留筛选后的重点变量，便于聚焦解释。
- `0 初始待办记录.ipynb`
  - 早期待办和思路记录，不属于当前正式主流程。
- `outputs_single_species_fe/`
  - 当前主线导出的单菌种长表结果。
- `outputs_compare_tables/`
  - 汇总后的横向对比表，适合比较不同菌种。
- `lancet_tables_by_species/`
  - 各菌种的 Lancet 风格结果表。

## 这个目录最适合回答的问题

- `R1xday` 与 `抗菌药物使用强度` 的正向信号，是否会在所有单菌种里同时出现？
- 哪些单菌种更像在响应温度、极端降雨、污染或卫生代理？
- 综合指标 `AMR_AGG_z` 的发现，是否主要由少数菌种拉动？
- 哪些菌种的方向与综合指标一致，哪些明显偏离？

## 当前应怎样解读这一层

这层的价值，不在于“再造一个新的主结论”，而在于提醒：

- 综合指标显著，不等于所有单菌种都显著；
- 严格双固定效应下，很多单菌种对核心变量的响应其实并不统一；
- 因此主线更适合写成“综合耐药风险的气候相关性分析”，而不是“所有耐药类型都同样响应气候变化”。

如果你需要一句更实用的话：

> 这里更适合用来界定结论边界，而不是替代主结论。

## 推荐查看顺序

1. 先打开 `1_VIF.ipynb`
   - 确认重点变量组合是否存在明显共线性问题。
2. 再看 `2_single_species.ipynb`
   - 理解逐菌种建模流程和输出结构。
3. 最后结合 `3_selected_X.ipynb` 与 `outputs_compare_tables/`
   - 横向比较各菌种对重点变量的响应。

## 常用输出怎么用

### `outputs_single_species_fe/`

适合做：

- 长表检索；
- 汇总显著性方向；
- 统计某个变量在多少个菌种中为正/为负/显著。

### `outputs_compare_tables/`

适合做：

- 多菌种横向比较；
- 补充材料表格；
- 回答“哪个菌种最受某类协变量影响”。

### `lancet_tables_by_species/`

适合做：

- 论文或汇报里的规范化表格引用；
- 对单菌种结果进行更正式的版式整理。
